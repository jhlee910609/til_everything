# 함수형 프로그래밍의 정의

## 함수형 프로그래밍의 핵심

- 순수함수
  - side effect 없음
  - 동일한 Input에 대해 항상 같은 output이 return 되어야함

```javascript
// 아래 함수는 순수함수
function add(a, b) {
  return a + b;
}
// 언제나 아래 인자를 전달한 함수를 호출할 경우, 10이 return됨
add(2, 5);

const obj1 = { val: 10 };
function add2(obj, b) {
  obj.val += b; // side effect 일으킴. 순수함수가 아님
}
// 아래 역시 순수함수가 아님
add2(obj1, 20);
```
