# 평가와 일급

## 평가

- 코드가 계산(Evalution) 되어 값을 만드는 것

## 일급

- 값으로 다룰 수 있음
- 변수에 담을 수 있음
- 함수의 인자로 사용될 수 있음
- 함수의 결과로 사용될 수 있음

## 일급 함수

- 함수를 값으로 다룰 수 있음
- 조합성(composition)과 추상화의 도구

```javascript
const add5 = (a) => a + 5;
console.log(add5);
console.log(add5(5)); // 10
```

## 고차 함수

- 함수를 인자로 받아 실행하는 함수
- 함수를 만들어 return 하는 함수(클로저를 만들어 리턴하는 함수)
